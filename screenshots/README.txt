# Instruções para Screenshots

Esta pasta contém arquivos de texto (.txt) que servem como placeholders para as capturas de tela do aplicativo ClimaSOS.

Para adicionar as imagens reais ao projeto:

1. Substitua cada arquivo .txt pelo arquivo de imagem correspondente (.png) com o mesmo nome base:
   - tela_principal_vazia.png
   - tela_principal_preenchida.png
   - tela_principal_multiplos_eventos.png
   - tela_identificacao.png

2. As imagens devem corresponder às seguintes telas:
   - tela_principal_vazia.png: Tela principal sem nenhum evento cadastrado
   - tela_principal_preenchida.png: Tela principal com um evento cadastrado
   - tela_principal_multiplos_eventos.png: Tela principal com vários eventos cadastrados
   - tela_identificacao.png: Tela de identificação mostrando os nomes e RMs dos alunos

3. Após adicionar as imagens, elas serão automaticamente exibidas no README.md do projeto.
